		=> Remove Package 
			(i) => Remove Package 
					sudo apt remove tree 
			(ii) => Remove a package and it's configuration files 
					sudo apt purge tree 
			(iii) => Clean up Unused Dependencies  
					sudo apt autoremove

					sudo apt-get clean
